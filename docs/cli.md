
# CLI

```
umaf-mini --help
```

- `-i, --input` input path or `-` for STDIN
- `-o, --output` output path or STDOUT
- `--json` output envelope JSON
- `--markdown` output normalized Markdown
- `--assume-type {txt|md|json}` for STDIN

Exit codes are stable and documented in `UMAFUserError.exitCode`.
